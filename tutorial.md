### Claude Installation
Source: 
[Claude Official Quickstart](https://code.claude.com/docs/en/quickstart)
- VSCode plugin
or
- Terminal: `curl -fsSL https://claude.ai/install.sh | bash`

For command line installation, you may see this note and follow along:
```
⚠ Setup notes:
  • Native installation exists but ~/.local/bin is not in your PATH. Run:

  echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc
```

### Quick Tips

- Add this game changer line at the end of your prompt: "Ask me clarifying questions, one at a time, until you're 95% confident you can complete the task."
- Update CLAUDE.md file on a daily basis!
